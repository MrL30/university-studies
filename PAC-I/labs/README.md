# 🐍 Python Laboratory Exercises

This directory contains all the weekly lab exercises completed during the **Applied Programming for Cybersecurity I (PAC-I)**. These labs were designed to build a strong foundation in Python programming, covering everything from basic syntax to more complex problem-solving.

## 📂 Directory Structure

Each folder corresponds to a specific practical class and contains the Python scripts (`.py`) with the solutions to the proposed exercises, as well as Jupyter Notebooks (`.ipynb`) sair exercises.

* [📁 `Lab01_extra`](./Lab01_extra) - Password Manager implementation using cryptography and regular expressions
* [📁 `Lab02`](./Lab02) - Basic syntax, variables, data types conversions, and simple math operations
* [📁 `Lab03`](./Lab03) - Control flow statements (`if`, `elif`, `else`), logical operators, and input validation
* [📁 `Lab04`](./Lab04) - Loops (`for`, `while`), string slicing, and introduction to functions
* [📁 `Lab05`](./Lab05) - Practical applications: Caesar Cipher, Change Calculator, and Number Guessing Game
* [📁 `Lab06`](./Lab06) - Tuples manipulation, unpacking, and Monte Carlo $\pi$ estimation
* [📁 `Lab07`](./Lab07) - Advanced lists, matrices, and Conway's Game of Life simulation
* [📁 `Lab08a`](./Lab08a) - Introduction to Object-Oriented Programming (Classes and Methods applied to a Bank Account)
* [📁 `Lab08b`](./Lab08b) - Advanced OOP concepts including Inheritance, Static Methods, and Class Variables
* [📁 `Lab09`](./Lab09) - Dictionaries, nested data structures, and data grouping for a Reservation System
* [📁 `Lab10`](./Lab10) - File handling operations, including reading and writing text, binary (`pickle`), CSV, and JSON files, as well as using context managers
* [📁 `Lab11`](./Lab11) - Log processing and data parsing from various file formats, including `.log` files, `.csv` metrics, and `.json` requests
* [📁 `Lab13`](./Lab13) - Advanced Exception Handling, including standard `try`/`except` blocks, creating custom exception hierarchies, and implementing custom context managers using `__enter__` and `__exit__`

## 🚀 How to executeTo run any specific exercise, navigate to the respective lab folder in your terminal and execute the Python file:

```bash
cd lab02
python3 exercise1.py
```
