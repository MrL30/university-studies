## 💻 Technologies & Tools

Throughout this course, I will be working with various technologies across Networking, Cloud Computing, Cybersecurity, and Software Development:

* **Programming & Scripting:** Python, JavaScript, Node.js, SQL, HTML/CSS
* **Operating Systems:** Linux (Server), Windows, macOS
* **Networking & Security:** TCP/IP, Firewalls, VPNs, IDS/IPS, Wireshark/tcpdump, Cisco/Routing
* **Cloud Computing:** AWS (EC2, S3, RDS, Lambda, DynamoDB), Virtualization
* **IoT & Hardware:** Arduino, Raspberry Pi, MQTT, RESTful APIs
